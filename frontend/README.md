To run the frontend folow these steps:

1. Install node package manager on your system and ensure you have npm / pnpm.
2. In the frontend directory, run:
    npm install 
    or
    pnpm install
3. On successful installation, run (in the same frontend directory):
    npm run dev
    or
    pnpm dev
4. Go to the configured localhost port and the frontend application should now be up and running!
